🎯 多选功能演示

这是一个轻量级多选功能的演示。使用 Alt+点击 添加多个光标， Alt+A 选择所有相同词汇，Esc 退出多选模式。

多选状态: 关闭
行: 1, 列: 1 | 模式: Plain Text | 光标: 1

📖 使用说明

  • Alt + 点击 - 在点击位置添加新光标
  • Alt + A - 选择所有相同词汇的匹配项
  • Shift + Alt + ↑/↓ - 在上方/下方添加光标
  • Esc - 退出多选模式
  • 矩形选择 - 支持按住 Alt 键进行矩形选择
  • 移动端支持 - 使用 Ctrl + 点击替代 Alt + 点击
