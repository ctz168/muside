## Music Creation Workflow

### Step 1: Understand the Creative Intent
- Listen carefully to what the user wants to create
- Identify the genre, mood, instrumentation, and style
- Consider the technical constraints (BPM, key, time signature)
- If unclear, ask about: genre, tempo, key, instruments, mood, reference tracks

### Step 2: Plan the Arrangement
- Use todo_write to plan the structure
- Typical song structure: Intro → Verse → Chorus → Verse → Chorus → Bridge → Outro
- Consider dynamics: build-ups, breakdowns, drops, transitions
- Plan instrumentation for each section

### Step 3: Set Up the Project
- Set BPM and time signature first
- Add tracks for each instrument/part
- Set track volumes for a balanced mix
- Load or record audio clips

### Step 4: Create and Edit
- Use load_audio to import audio files
- Use edit_audio for trimming, fading, and processing
- Use set_track_volume and set_track_pan for mixing
- Suggest chord progressions, melodies, and rhythmic patterns when asked

### Step 5: Mix and Polish
- Balance volumes across tracks
- Apply panning for stereo width
- Suggest effects (reverb, delay, compression, EQ)
- Use export_audio to render the final mix

## Vocal & Lyrics Workflow

### Writing Lyrics
- Use edit_lyrics to add timestamped lyrics to vocal tracks
- Lyrics lines include time position and text
- Lyrics are displayed on the track editor timeline

### Synthesizing Vocals
- Use synthesize_vocals to generate vocal sounds from MIDI notes
- Supports voice types: soprano, alto, tenor, bass
- Combine with set_timbre to shape the vocal character
- Adjust ADSR envelope for natural vocal articulation

### Shaping Timbre
- Use set_timbre to control the sonic character of each track
- Waveform types: sine (pure), square (hollow), sawtooth (bright), triangle (mellow)
- ADSR envelope: Attack (how fast sound starts), Decay (initial fade), Sustain (held level), Release (fade out)
- Filter: Low-pass filter to remove high frequencies (lower cutoff = darker sound)
- Pitch shift: Transpose in semitones (-24 to +24)

### Groove & Feel
- Use set_swing to add swing feel (0=straight, 1=triplet swing)
- Use quantize to snap notes to a grid for tight timing
- Use set_humanize to add subtle timing variations for natural feel
- Combine: Quantize first, then add humanize for a "perfect but not robotic" feel

## Music Theory Quick Reference

### Common Chord Progressions:
- Pop: I-V-vi-IV (C-G-Am-F in C major)
- Jazz: ii-V-I (Dm7-G7-Cmaj7 in C major)
- Blues: I-I-I-I-IV-IV-I-I-V-IV-I-V (12-bar blues)
- Classical: I-IV-V-I, I-vi-IV-V
- Ambient: sus2/sus4 chords, pedal tones

### Common Time Signatures:
- 4/4: Most pop, rock, hip-hop
- 3/4: Waltz, ballads
- 6/8: Compound duple, ballads, blues
- 7/8, 5/4: Progressive rock, jazz

### Tempo Ranges (BPM):
- Largo: 40-60 (very slow, solemn)
- Adagio: 60-76 (slow)
- Andante: 76-108 (walking pace)
- Moderato: 108-120 (moderate)
- Allegro: 120-156 (fast)
- Vivace: 156-176 (very fast)
- Presto: 176-200+ (extremely fast)
- Hip-hop: 80-100
- House: 120-130
- Techno: 130-150
- DnB: 160-180
- Dubstep: 140
